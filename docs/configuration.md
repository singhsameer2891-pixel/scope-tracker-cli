# Configuration Reference

## `scope_tracker_config.json`

Created by `scope-tracker init`. Located in the `scope-tracker/` directory.

### `global_settings`

| Field | Type | Default | Description |
|---|---|---|---|
| `reporting_slack_channel` | string | `"scope-tracker"` | Slack channel where run reports are posted. No `#` prefix. |
| `reporting_slack_last_read` | string \| null | `null` | Timestamp of the last message read in the reporting channel. Updated automatically after each report post. |
| `default_timezone` | string | `"Asia/Kolkata"` | IANA timezone string used for timestamps in the sheet and reports. |

### `sheet_config`

| Field | Type | Default | Description |
|---|---|---|---|
| `uat_rounds` | integer | `5` | Number of UAT rounds. Each round adds a Status + Notes column pair. Max 10. |
| `status_options` | string[] | `["To be tested", "Passed", "Passed with iteration", "Failed", "Blocked"]` | Dropdown options for UAT Status columns. |
| `version_options` | string[] | `["LIVE", "Next release", "Parked", "Fast follower"]` | Dropdown options for Target Version column. |
| `scope_decision_options` | string[] | `["In Scope", "Fast Follower", "Pushed to V2", "Parked", "Active Blocker", "Conflicting Signal"]` | Dropdown options for Scope Decision column. |
| `blocker_options` | string[] | `["Yes", "No"]` | Dropdown options for Blocker? column. |
| `prd_identifier_column_names` | string[] | `["ID", "Identifier", "#", "Ref"]` | Header names the tool uses to find the identifier column in the PRD table. |
| `prd_story_column_names` | string[] | `["User Story", "Story", "Feature", "Requirement", "Description"]` | Header names the tool uses to find the story/feature column in the PRD table. |

### `google_sheets`

| Field | Type | Description |
|---|---|---|
| `client_secret_path` | string | Absolute path to the Google OAuth2 `client_secret.json` file. Set during `scope-tracker init`. Used for direct Google Sheets API access. |

The OAuth2 token (`token.json`) is saved in the `scope-tracker/` directory after the first successful consent flow and is automatically refreshed on subsequent runs. It is gitignored.

### `projects[]`

Each entry in the `projects` array represents one tracked project.

| Field | Type | Description |
|---|---|---|
| `name` | string | Project name. Used as folder name. Lowercase, no spaces. |
| `enabled` | boolean | Whether this project is included in `scope-tracker run`. |
| `folder` | string | Folder name under `scope-tracker/` for this project's data. Usually same as `name`. |
| `slack_channel` | string | Slack channel to monitor for scope decisions. No `#` prefix. |
| `sheet_url` | string | Google Sheet URL. Populated automatically by `init-sheet`. Empty until then. |
| `prd_source.type` | string | PRD source type: `"google-drive"`, `"confluence"`, or `"none"`. |
| `prd_source.url` | string | URL to the PRD document. Empty if type is `"none"`. |
| `prd_source.last_modified` | string \| null | Last known modified timestamp of the PRD. Updated after each read. |
| `slack_last_run_timestamp` | string \| null | Slack watermark timestamp. Messages after this are considered new. |
| `run_count` | integer | Number of pipeline runs completed for this project. |
| `last_run_date` | string \| null | ISO date of the last run. |

### Example

```json
{
  "global_settings": {
    "reporting_slack_channel": "scope-tracker",
    "reporting_slack_last_read": null,
    "default_timezone": "Asia/Kolkata"
  },
  "google_sheets": {
    "client_secret_path": "/Users/you/credentials/client_secret.json"
  },
  "sheet_config": {
    "uat_rounds": 5,
    "status_options": ["To be tested", "Passed", "Passed with iteration", "Failed", "Blocked"],
    "version_options": ["LIVE", "Next release", "Parked", "Fast follower"],
    "scope_decision_options": ["In Scope", "Fast Follower", "Pushed to V2", "Parked", "Active Blocker", "Conflicting Signal"],
    "blocker_options": ["Yes", "No"],
    "prd_identifier_column_names": ["ID", "Identifier", "#", "Ref"],
    "prd_story_column_names": ["User Story", "Story", "Feature", "Requirement", "Description"]
  },
  "projects": [
    {
      "name": "scalper",
      "enabled": true,
      "folder": "scalper",
      "slack_channel": "scalper",
      "sheet_url": "https://docs.google.com/spreadsheets/d/...",
      "prd_source": {
        "type": "google-drive",
        "url": "https://docs.google.com/document/d/...",
        "last_modified": null
      },
      "slack_last_run_timestamp": null,
      "run_count": 0,
      "last_run_date": null
    }
  ]
}
```

## `.mcp.json`

Auto-generated by `scope-tracker init`. Located in the `scope-tracker/` directory. **Never commit this file to git** — it contains credentials.

### Structure

```json
{
  "mcpServers": {
    "slack": { ... },
    "gdrive": { ... },
    "confluence": { ... }
  }
}
```

### `slack` (always present)

| Field | Value |
|---|---|
| `command` | `"npx"` |
| `args` | `["-y", "@modelcontextprotocol/server-slack"]` |
| `env.SLACK_BOT_TOKEN` | Your Slack Bot Token (starts with `xoxb-`) |
| `env.SLACK_TEAM_ID` | Your Slack Team ID (starts with `T`) |

Create a Slack bot at [api.slack.com/apps](https://api.slack.com/apps).

### `gdrive` (present if any project uses Google Drive)

| Field | Value |
|---|---|
| `command` | `"npx"` |
| `args` | `["-y", "@modelcontextprotocol/server-gdrive"]` |
| `env.GDRIVE_CREDENTIALS_FILE` | Absolute path to your Google OAuth2 credentials JSON file |

Create credentials at [console.cloud.google.com](https://console.cloud.google.com) → APIs & Services → Credentials → Create OAuth client ID → Desktop app.

### `confluence` (present if any project uses Confluence)

| Field | Value |
|---|---|
| `command` | `"npx"` |
| `args` | `["-y", "@aashari/mcp-server-atlassian-confluence"]` |
| `env.ATLASSIAN_SITE_NAME` | Your Atlassian site name (e.g. `yourteam` from `yourteam.atlassian.net`) |
| `env.ATLASSIAN_USER_EMAIL` | Your Atlassian user email |
| `env.ATLASSIAN_API_TOKEN` | Your Atlassian API token |

Create an API token at [id.atlassian.com](https://id.atlassian.com) → Security → API tokens.

### Example

```json
{
  "mcpServers": {
    "slack": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-slack"],
      "env": {
        "SLACK_BOT_TOKEN": "xoxb-1234567890-abcdefghijk",
        "SLACK_TEAM_ID": "T01234ABCDE"
      }
    },
    "gdrive": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-gdrive"],
      "env": {
        "GDRIVE_CREDENTIALS_FILE": "/Users/you/credentials/google-oauth.json"
      }
    }
  }
}
```

## Updating Configuration

### Changing dropdown options

Edit the relevant array in `scope_tracker_config.json` (e.g. `status_options`, `scope_decision_options`) and run `scope-tracker run`. The new options will be applied to the sheet on the next run.

### Adding UAT rounds

Increase `sheet_config.uat_rounds` in `scope_tracker_config.json`. New columns are added on the next sheet update. Existing columns are never removed.

### Disabling a project

Set `"enabled": false` in the project's config entry. The project will be skipped by `scope-tracker run` but its data is preserved.
